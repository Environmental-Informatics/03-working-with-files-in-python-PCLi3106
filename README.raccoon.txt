The name of the process script is Evaluate_Raccoon_Life.py

This script is written for reading a comma separated text file, which contains the record of behavior of a raccoon whose name is George.

Input: 2008Male0006.txt
Output: Georges_life.txt

The name, average location, distance traveled, average energy level, and the end date of the raccoon, George, is recorded. 

The distance of movement is obtained from the location recorded in the input. The average function is applied to the distance, location, and energy level.


The behavior and characteristics of George is recorded in a test file with new format: str Line out put as a header block of basci information and the new tab-separated part 
for the raw data we read.
